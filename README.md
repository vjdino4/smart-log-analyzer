# 🔐 Smart Log Analyzer & Intrusion Detection Dashboard

## 📌 Project Overview
This project simulates a **Mini Security Operations Center (SOC)** system built using Python.

The system analyzes authentication logs, detects suspicious activity such as brute force attacks, and visualizes security events through a dashboard.

This project demonstrates how security analysts process logs and detect threats using automated tools.

---

## ⚙️ Features

- Log parsing using Python
- Detection of repeated failed login attempts
- Brute force attack identification
- Automated security report generation
- Security event visualization dashboard

---

## 🏗 Project Architecture

logs.txt → log_analyzer.py → security_report.txt → dashboard.py

1. Logs are collected from a simulated system log file
2. The analyzer script processes the logs
3. Suspicious activity is detected
4. Results are saved into a security report
5. Dashboard visualizes security events
